Functions by the PL calculator


The calculator supports operators: +, -, multiply '*', and divide '/'
Language switching commands: rus, eng
The 'M' command allows you to insert the last obtained result into the calculation,
which is automatically saved. 'C + Enter' - clear memory.
The 'St' command raises the penultimate number to the power of the last number. For
xample, '2 5 st' (two to the fifth power) will give you 32.00
The 'root' command extracts the square root of the last number.
The '%' command extracts the percentage of the penultimate number equal to the last
number. For example, '120 10 %' will return 12.00
The 'info' command displays the contents of the State class on the screen.
The 'S' command saves the current data to a document.
The 'sm' command switches between the Array / List data structure method.
The 'toRom' command converts the memory content into Roman numerals.
The 'toBin' command converts the memory content into binary number.
The 'toOct' command converts the memory content into octal number.
The 'toHex' command converts the memory content into hexadecimal number.
The calculator performs operations with Roman, binary, octal, and hexadecimal
numbers.
For octal numbers, the prefix '0o' is mandatory, for hexadecimal '0x',
and for binary '0b'
To automatically output all results in the desired numbering format, use the
commands: outDec (decimal), outBin (binary), outOct (octal), outHex (hexadecimal),
ourRome (Roman).
For switch logger method use command logPl (simple) or logSl (slf4j)
Exit the program: 'E + Enter'.

Функции калькулятора


Калькулятор поддерживает операторы: +, -, умножить '*' и разделить '/' >Команды переключения языка: rus, eng.
Команда 'M' позволяет вставить в вычисление последний полученный результат, который сохраняется автоматически. 'C + Enter' - очистить память.
Команда 'St' - возводит предпоследнее число в степень соответствующую последнему числу. Например, '2 5 st' (два в пятой степени) даст 32.00
Команда 'root' - извлечет квадратный корень из последнего числа.
Команда '%' - извлечет из предпоследнего числа проценты, количество которых равно последнему числу. Например, '120 10 %' вернет 12.00
Команда 'S' - Сохраняет текущие данные в документ.
Команда 'sm' - переключает метод структуры данных Массив / Строка.
Команда 'toRom' - конвертирует содержимое памяти в римские числа.
Команда 'toBin' - конвертирует содержимое памяти в двоичное число.
Команда 'toOct' - конвертирует содержимое памяти в восьмеричное число.
Команда 'toHex' - конвертирует содержимое памяти в шестнадцатеричное число.
Калькулятор выполняет операции с римскими, двоичными, восьмеричными и шестнадцатеричными числами.
Для восьмеричных чисел обязателен префикс '0o', шестнадцатеричных '0x', двоичных '0b'.
Для вывода всех результатов в нужном формате исчисления автоматически, используйте команды:
outDec (десятеричные), outBin (двоичные), outOct (восьмеричные), outHex (шестнадцатеричные), ourRome (римские).
Для переключения логгера используйте команды logPl (simple) or logSl (slf4j).
Выход из программы: 'E + Enter'.
