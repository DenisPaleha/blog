Функции поддерживаемые калькулятором


Калькулятор поддерживает операторы: +, -, умножить '*' и разделить '/' >Команды переключения языка: rus, eng.
Команда 'M' позволяет вставить в вычисление последний полученный результат, который сохраняется автоматически. 'C + Enter' - очистить память.
Команда 'St' - возводит предпоследнее число в степень соответствующую последнему числу. Например, '2 5 st' (два в пятой степени) даст 32.00
Команда 'root' - извлечет квадратный корень из последнего числа.
Команда '%' - извлечет из предпоследнего числа проценты, количество которых равно последнему числу. Например, '120 10 %' вернет 12.00
Команда 'S' - Сохраняет текущие данные в документ.
Команда 'sm' - переключает метод структуры данных Массив / Строка.
Команда 'toRom' - конвертирует содержимое памяти в римские числа.
Команда 'toBin' - конвертирует содержимое памяти в двоичное число.
Команда 'toOct' - конвертирует содержимое памяти в восьмеричное число.
Команда 'toHex' - конвертирует содержимое памяти в шестнадцатеричное число.
Калькулятор выполняет операции с римскими, двоичными, восьмеричными и шестнадцатеричными числами.
Для восьмеричных чисел обязателен префикс '0o', шестнадцатеричных '0x', двоичных '0b'.
Для вывода всех результатов в нужном формате исчисления автоматически, используйте команды:
outDec (десятеричные), outBin (двоичные), outOct (восьмеричные), outHex (шестнадцатеричные), ourRome (римские).
Для переключения логгера используйте команды logPl (simple) or logSl (slf4j).
Выход из программы: 'E + Enter'.
